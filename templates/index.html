To deploy a simple Flask app with Docker:

  1. Write your Flask app in app.py.
  2. Create a Dockerfile to containerize your app.
  3. Build the Docker image: docker build -t flask-docker-app .
  4. Run the container: docker run -p 5000:5000 flask-docker-app

Visit http://localhost:5000 to view your app.
